
# Focus Timer ⏳


## Funcionalidade

O Focus Timer é um cronômetro com a funcionalidade de alarme aplicada quando o tempo definido se esgotar.

Também há uma música de fundo funcionando em looping, podendo ser mutada a qualquer momento.
